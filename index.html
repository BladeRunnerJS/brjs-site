Divide & conquer your web apps

BladeRunnerJS is a development toolkit and lightweight framework for building web applications consisting of one or more components called Blades. It comes complete with some seriously useful tools which make it easy to develop, test and deploy your app.

Blades

{% include docs/concepts/blade_overview.html %}

Workbenches

Workbenches allow Blades to be developed, debugged and run independently of the app.

Workbench Widgets allow you to interact with the Blade through stubbed services and via the event-hub. For example you can view your Blade in different states by sending it mock events via a mock events Workbench Widget.

Clean Code

Code is organized cleanly using classes, with one class per file, and JavaScript interface support is provided to help with creating large scalable code-bases.

Services are a first-class concept that allow your application to be written before the back-end is finished, and can be replaced by mocks in your tests.
